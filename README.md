# 💰 Personal Expenses Tracker

A data analysis project built using **R** to analyze personal spending habits, clean financial data, visualize expenses, and generate meaningful insights.

---

## 📖 Overview

Managing personal finances is easier when you can visualize where your money goes. This project performs exploratory data analysis on personal expense data using R and produces visualizations and statistical summaries to help understand spending patterns.

---

## ✨ Features

- 📂 Import expense dataset
- 🧹 Data cleaning and preprocessing
- ❌ Handle missing values
- 📊 Category-wise expense analysis
- 📈 Daily expense trend visualization
- 🥧 Payment mode distribution
- 📉 Histogram of expenses
- 📌 Summary statistics
- 📐 Simple Linear Regression analysis

---

## 🛠️ Tech Stack

- **Language:** R
- **Libraries:**
  - ggplot2
  - dplyr

## 📊 Dataset

The dataset contains personal expense records including:

| Column | Description |
|---------|-------------|
| Date | Transaction Date |
| Category | Expense Category |
| Amount | Expense Amount |
| Payment_Mode | Cash / UPI / Card etc. |

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Konchigeriakash/Personal-Expenses-Tracker.git
cd Personal-Expenses-Tracker
```

### Install Required Packages

```R
install.packages(c("ggplot2", "dplyr", "lubridate"))
```

### Run the Project

```R
source("analysis.R")
```
